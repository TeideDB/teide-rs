Data Types

Teide supports numeric, string, boolean, and temporal data types.

Numeric Types

SQL Type Aliases Storage Description
INTEGER INT, INT4 32-bit Signed integer
BIGINT INT8, INT64 64-bit Large signed integer
REAL DOUBLE, FLOAT, DOUBLE PRECISION 64-bit Floating-point number

String Types

SQL Type Aliases Storage Description
VARCHAR TEXT, STRING Variable UTF-8 text (stored as interned symbols)

Boolean Type

SQL Type Aliases Storage Description
BOOLEAN BOOL 1-bit true or false

Temporal Types

SQL Type Aliases Storage Description
DATE 32-bit Calendar date (days since 2000-01-01)
TIME 64-bit Time of day
TIMESTAMP 64-bit Date and time (microseconds since 2000-01-01)

Type Casting

-- CAST syntax
SELECT CAST(42 AS REAL);

-- PostgreSQL :: shorthand
SELECT 42::REAL;
